# fulldraw

A drawing software.

drawing to fullscreen canvas.

No UI, all white.

## for

* wacom pentablet
* Windows 10

## how to build

### require

install wacom driver. (to will appear `wintab32.dll`)

get wintab header files.

* msgpack.h
* pktdef.h
* wintab.h

### command

1. `"C:\Program Files (x86)\Microsoft Visual Studio\2017\Community\Common7\Tools\VsDevCmd.bat"`
2. `nmake`
