# fulldraw

A drawing software.

Draw to fullscreen canvas.

UI is none, all white.

## for

* wacom pentablet
* Windows 10

## usage

### keyboard

* `↑` `↓`: tweak pen size
* `←` `→`: tweak pressure sens

## how to build

### command

1. `"C:\Program Files (x86)\Microsoft Visual Studio\2017\Community\Common7\Tools\VsDevCmd.bat"`
2. `nmake`
